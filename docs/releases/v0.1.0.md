## OpenCode Snapshot Tool v0.1.0

First public Windows build of the standalone OpenCode snapshot inspector and cleanup tool.

### Highlights

- Scans legacy and current OpenCode Git snapshot layouts.
- Maps snapshot trees to OpenCode session metadata without modifying `opencode.db`.
- Shows repository, snapshot, retention, and filesystem-size information before cleanup.
- Protects retained trees before Git/LFS pruning and requires an explicit preview plus confirmation.
- Uses the Bauhaus / Neo-Brutalist interface defined in `design.md`, with bundled Space Grotesk and Inter fonts and high-contrast custom controls.

### Download

Download `OpenCode-Snapshot-Tool-v0.1.0-windows-x64.zip`, extract it, and run `opencode-snapshot-tool.exe`.

Scanning and previewing are read-only. Cleanup is destructive; close OpenCode or ensure it is idle before confirming cleanup.
