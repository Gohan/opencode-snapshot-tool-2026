## OpenCode Snapshot Tool v0.1.1

### Interface update

- Replaces the system title bar with Bauhaus / Neo-Brutalist application chrome, including custom drag, minimize, maximize/restore, close, border, and resize behavior.
- Makes the cleanup safety gate explicit as a two-step batch flow across all scanned repositories.
- Step 01 is a read-only batch preview and shows its repository coverage.
- Step 02 stays visibly locked until a valid preview exists, then shows the exact number of trees ready for release.
- Settings changes continue to invalidate the reviewed plan before any destructive action can run.

Download `OpenCode-Snapshot-Tool-v0.1.1-windows-x64.zip`, extract it, and run `opencode-snapshot-tool.exe`.

SHA-256: `06e9f50f84e67adfe46686b5304a20a515b7754f2990580e47ae034c51d3038d`

Scanning and previewing are read-only. Batch cleanup is destructive and still requires a dedicated confirmation dialog.
